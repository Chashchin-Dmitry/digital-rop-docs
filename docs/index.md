## Возможности платформы

<div class="grid cards" markdown>

-   :material-phone-check:{ .lg .middle } **Автоматическая оценка звонков**

    ---

    ИИ проверяет каждый разговор по настроенному чек-листу (скрипту продаж)

-   :material-chart-line:{ .lg .middle } **Отслеживание конверсии**

    ---

    Система определяет, привёл ли звонок к результату

-   :material-school:{ .lg .middle } **Формирование тестов**

    ---

    Автоматическое создание обучающих тестов на основе ошибок менеджеров

-   :material-table-large:{ .lg .middle } **Кастомная аналитика**

    ---

    Извлечение любой информации из звонков через настраиваемые промты

</div>

## Быстрый старт

1. [Зарегистрируйтесь](start/registration.md) через Telegram-бота
2. Подключите источник звонков (CRM, телефонию или Яндекс Диск)
3. Настройте [шаблоны скриптов](settings/scripts-templates.md) для оценки
4. Привяжите скрипты к [воронкам и стадиям](settings/scripts-prompts.md)

После этого система начнёт автоматически анализировать все поступающие звонки.
