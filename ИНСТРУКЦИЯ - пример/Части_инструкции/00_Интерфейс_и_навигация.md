# Интерфейс и навигация

## Главное меню и навигация

### Левая панель навигации

**Структура разделов**:

#### АНАЛИТИКА
- **Коммуникации** - просмотр и анализ звонков
- **Менеджеры** - статистика по сотрудникам  
- **Тесты** - управление тестированием

#### ГРАФИКИ
- **Менеджеры** - визуализация производительности
- **Оценка** - графики качества звонков
- **Статус тестов** - статистика прохождения тестов
- **Динамика тестов** - временные графики тестирования

#### ТАБЛИЦЫ
- Список созданных кастомных таблиц
- Настраиваемая аналитика с ИИ-промтами

#### ДРУГОЕ  
- **Чат с ИИ** - интерактивные запросы к системе
- **Статистика файлов** - информация об обработанных звонках

#### НАСТРОЙКИ
- **Подключение** - API ключи и технические параметры
- **Скрипты и промты** - настройка чек-листов
- **Офисы и руководители** - структура организации
- **Таблицы** - создание кастомных полей
- **Пользователи** - управление доступом

#### ДОП. НАСТРОЙКИ
- **Слова-паразиты** - настройка фильтрации речи

### Индикация текущего местоположения

**Активный раздел**:
- Подсвечивается **зеленым цветом** в левой панели
- Название раздела отображается зеленым шрифтом
- Неактивные разделы отображаются черным шрифтом

**Хлебные крошки**: В текущей версии отсутствуют

### Переходы между разделами

**Основной способ навигации**:
1. Нажмите на нужный раздел в левой панели
2. Система мгновенно переключится на выбранный раздел
3. Новый раздел будет подсвечен зеленым

**Внутренняя навигация**:
- В разделах с большим количеством данных используется пагинация
- Фильтры для уточнения отображаемой информации
- Сортировка по различным параметрам

## Работа с данными

### Просмотр списков

**Общие принципы**:
- Данные отображаются в табличном виде
- Поддержка сортировки по столбцам
- Пагинация для больших объемов данных

### Детальный просмотр

**Раскрытие элементов**:
- Нажатие на элемент списка открывает детальную информацию
- Возврат к списку через повторное нажатие или навигацию

### Переключение видов

**Менеджеры. Переключиться на вид таблица.png**

В разделе "Менеджеры" доступны два вида отображения:
- **Карточный вид** - наглядное представление данных
- **Табличный вид** - компактное отображение в таблице

**Менеджеры. вид Вид таблица.png**

## Особенности интерфейса

### Цветовая индикация

**Зеленый цвет**:
- Активный раздел в навигации
- Успешно выполненные этапы
- Положительные показатели

**Красный цвет**:
- Невыполненные этапы
- Ошибки и проблемы
- Критические показатели

### Адаптивность

**Масштабирование**:
- Интерфейс адаптируется под размер экрана
- Изображения и графики масштабируются автоматически
- Таблицы поддерживают горизонтальную прокрутку при необходимости

### Доступность функций по ролям

**Видимость разделов**:
- **Менеджеры** - ограниченный доступ к аналитике и графикам
- **Руководители** - доступ к данным своего офиса + управление тестами
- **Администраторы** - полный доступ ко всем разделам и настройкам

**Важно**: Разделы "Настройки" и "Доп. настройки" доступны только администраторам и супер-администраторам.
