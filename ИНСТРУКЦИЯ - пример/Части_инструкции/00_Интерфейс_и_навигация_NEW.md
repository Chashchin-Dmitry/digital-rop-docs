# Интерфейс и навигация

**Структура разделов.png** - Основное меню навигации системы "Цифровой РОП"

Система имеет интуитивную структуру навигации с левой панелью меню и контентной областью для отображения данных. Обеспечивает быстрый доступ ко всем функциям системы в зависимости от роли пользователя.

## Левая панель навигации

**Структура разделов.png**

{INTERFACE} Левая панель содержит древовидную структуру разделов, сгруппированных по функциональным блокам. Активный раздел выделяется зеленым цветом, неактивные отображаются черным. Панель остается видимой на всех страницах системы для быстрой навигации.

### Блок "Аналитика"

{INTERFACE} Включает три основных раздела для анализа данных: "Коммуникации" для просмотра звонков с оценками, "Менеджеры" для статистики сотрудников с рейтингами и "Тесты" для управления обучающими материалами. Каждый раздел открывается одним кликом.

### Блок "Графики"

{INTERFACE} Содержит четыре раздела визуализации: "Менеджеры" с динамикой работы сотрудников, "Оценка" с общей статистикой системы, "Статус тестов" с распределением тестов по статусам и "Динамика тестов" с временными трендами прохождения.

### Блок "Таблицы"

{INTERFACE} Отображает список созданных пользовательских таблиц с настраиваемой аналитикой. Каждая таблица создается администратором с индивидуальными промтами для извлечения специфической информации из звонков.

### Блок "Другое"

{INTERFACE} Включает вспомогательные инструменты: "Чат с ИИ" для интерактивных запросов к системе анализа и "Статистика файлов" для мониторинга технического состояния обработки аудиозаписей.

### Блок "Настройки"

{TECHNICAL} Доступен только администраторам и содержит пять разделов конфигурации: "Подключение" с API параметрами, "Скрипты и промты" для настройки чек-листов, "Офисы и руководители" для структуры организации, "Таблицы" для создания пользовательских полей и "Пользователи" для управления доступом.

### Блок "Дополнительные настройки"

{TECHNICAL} Содержит раздел "Слова-паразиты" для настройки автоматического выделения нежелательных слов в транскрипциях звонков. Слова настраиваются администратором и подсвечиваются красным цветом в текстах разговоров.

## Индикация текущего местоположения

{INTERFACE} Система использует цветовую индикацию для отображения активного раздела. Выбранный раздел в левой панели подсвечивается зеленым цветом и зеленым шрифтом. Все остальные разделы отображаются черным цветом для четкого различения.

## Переходы между разделами

{INTERFACE} Навигация осуществляется одним кликом по названию раздела в левой панели. Система мгновенно загружает выбранный раздел без перезагрузки страницы. Новый активный раздел автоматически подсвечивается зеленым цветом.

## Фильтрация и поиск

### Фильтры по дате

**Коммуникации. Фильтр по дате.png**

{INTERFACE} Календарный интерфейс позволяет выбрать начальную и конечную дату для анализа данных. По умолчанию система отображает данные за последнюю неделю. Фильтр применяется автоматически после выбора периода.

### Фильтры по сотрудникам

**Коммуникации. Фильтр по менеджеру.png**

{INTERFACE} Выпадающий список содержит всех активных менеджеров системы с поддержкой автодополнения при вводе фамилии. Позволяет быстро найти конкретного сотрудника в больших коллективах.

### Фильтры по офисам

**Коммуникации. Фильтр по офису.png**

{TECHNICAL} Фильтрация доступна только руководителям офисов и администраторам. Список включает все офисы компании с указанием количества активных менеджеров в каждом подразделении. При выборе офиса данные автоматически ограничиваются сотрудниками выбранного подразделения.

## Переключение режимов отображения

**Менеджеры. Переключиться на вид таблица.png**

{INTERFACE} В разделе "Менеджеры" доступны два режима просмотра данных: карточный вид для наглядного представления статистики каждого сотрудника и табличный вид для компактного отображения всей информации с возможностью сортировки.

## Права доступа к разделам

{TECHNICAL} Видимость разделов определяется ролью пользователя: менеджеры видят только разделы "Аналитика" и ограниченные "Графики", руководители офисов получают дополнительный доступ к управлению тестами, администраторы имеют полный доступ включая "Настройки" и "Дополнительные настройки".
