# Что такое Цифровой РОП

**Версия инструкции:** 1.0  
**Дата создания:** Январь 2025  
**Система:** Цифровой РОП (on-premise установка Setl Group)  
**Заказчик:** [Setl Group](https://setlgroup.ru)  
**Разработчик:** [BVMax](https://bvmax.ru)

Система **Цифровой РОП** представляет собой комплексное решение для автоматической оценки телефонных звонков с использованием искусственного интеллекта. Система анализирует все разговоры [менеджеров](#аналитика-менеджеры) по уникальным скриптам и [чек-листам](#настройки-скрипты-и-промты) компании.

## Как работает система

{TECHNICAL} Процесс обработки звонка включает получение аудиофайлов из CRM или телефонии, транскрибацию речи в текст, разделение спикеров (менеджер/клиент), ИИ-анализ по корпоративному скрипту и формирование результата с баллами, комментариями, рекомендациями и [тестами](#аналитика-тесты) для обучения.

## Ключевые возможности

### Автоматическая оценка звонков
- Анализ по настраиваемым чек-листам компании
- Оценка каждого этапа: +1 балл (выполнен) или -1 балл (не выполнен)  
- Процентная оценка выполнения каждого этапа по всем звонкам

### Система обучения менеджеров
{INTERFACE} На основе выявленных ошибок система автоматически создает персонализированные тесты для каждого менеджера. Тесты формируются из невыполненных этапов чек-листа и требуют одобрения руководителя перед отправкой сотруднику.

- [Детальная аналитика](#аналитика-коммуникации) по каждому звонку
- [Статистика работы менеджеров](#аналитика-менеджеры) и их рейтинги
- [Графики и визуализация](#графики-менеджеры) динамики показателей
- [Настраиваемые таблицы](#таблицы) для кастомной аналитики

### Интеграция с корпоративными системами
{TECHNICAL} Система поддерживает API для получения звонков из CRM, отправки результатов анализа обратно в корпоративные системы, интеграцию с LDAP для управления пользователями и настройку webhook для автоматической передачи данных.

## Роли пользователей

{INTERFACE} Система поддерживает четыре типа ролей: Менеджеры (просмотр собственных звонков и прохождение тестов), Руководители офисов (данные своего офиса, одобрение тестов), Администраторы (полный доступ, настройка системы) и Супер-администратор (управление всеми аспектами системы).

## Преимущества использования

- **Объективная оценка** - исключение субъективности в оценке работы
- **Автоматизация процессов** - снижение нагрузки на руководителей
- **Персонализированное обучение** - тесты на основе реальных ошибок
- **Масштабируемость** - обработка больших объемов звонков
- **Интеграция** - работа с существующими корпоративными системами
