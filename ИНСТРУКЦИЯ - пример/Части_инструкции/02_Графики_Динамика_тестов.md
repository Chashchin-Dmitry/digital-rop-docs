# 2. Графики - Динамика тестов

Раздел **Графики - Динамика тестов** предоставляет визуальный анализ процесса создания и прохождения [тестов](#аналитика-тесты) менеджерами. Позволяет отслеживать эффективность системы обучения и контролировать процесс развития сотрудников.

## Общий вид

**Динамика тестов. Общий вид .png** - Основной экран с ключевыми метриками и графиками

{INTERFACE} Страница содержит верхние метрики (тестов за неделю, за месяц, среднее на менеджера), основной график "Динамика прохождения тестов за месяц" и дополнительный график "Статус тестов по неделям за месяц". Все данные обновляются в реальном времени на основе активности менеджеров.

## Динамика прохождения тестов

**Динамика тестов. Динамика прохождения тестов за выбранный месяц.png** - График прохождения тестов по дням

{TECHNICAL} График отображает количество пройденных тестов по дням месяца. Ось X показывает дни (1-30/31), ось Y - количество тестов. Позволяет выявить пики активности, периоды спада и оценить общую динамику обучения сотрудников в выбранном временном периоде.

## Статус тестов по неделям

**Динамика тестов. Статус тестов по неделям за выбранный месяц.png** - Распределение тестов по статусам

{INTERFACE} Столбчатая диаграмма показывает распределение тестов по статусам (Не одобрен, Не пройден, Пройден) в разрезе недель месяца. Каждый столбец представляет неделю, цветовая кодировка отражает статусы тестов для быстрого визуального анализа.

## Фильтры

**Динамика тестов. Фильтры по офису и менеджерам.png** - Панель настройки отображения

{INTERFACE} Система фильтрации включает выбор временного периода (год и месяц), фильтрацию по [офисам](#настройки-офисы-и-руководители) и конкретным [менеджерам](#аналитика-менеджеры). Позволяет настроить отображение данных для детального анализа конкретных подразделений или сотрудников.
