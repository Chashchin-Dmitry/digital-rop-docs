# 2. Графики - Менеджеры

Раздел **Графики - Менеджеры** обеспечивает визуализацию эффективности работы [менеджеров](#аналитика-менеджеры) в динамике. Позволяет отслеживать тренды выполнения [скриптов](#настройки-скрипты-и-промты) и конверсии в разрезе времени.

## Общий вид

**Менеджеры-гр. Общий вид.png** - Комбинированный график среднего по менеджерам

{INTERFACE} График "Среднее по менеджерам" показывает столбцы с процентом выполнения скрипта по каждому менеджеру, линию с количеством обработанных звонков и две оси Y (левая для процентов 0-100%, правая для количества звонков). Данные отображаются по каждому менеджеру за выбранный месяц.

## Динамика соблюдения скрипта

**Менеджеры-гр. Динамика соблюдения скрипта по менеджерам за выбранный месяц.png** - Ежедневная динамика выполнения скрипта

{TECHNICAL} График отображает временную динамику по дням месяца (ось X) и баллам от положительных до отрицательных (ось Y). Каждый менеджер представлен отдельной линией с легендой справа, включающей список всех менеджеров с цветовой кодировкой. Позволяет анализировать пики и провалы в работе, общие тенденции по дням и сравнительную динамику между менеджерами.

## Конверсия менеджеров

**Менеджеры-гр. Конверсия менеджеров за выбранный месяц.png** - Динамика конверсии по дням

{INTERFACE} Аналогичная визуализация для отслеживания конверсии: ось X показывает дни месяца, ось Y - процент конверсии (0-100%), линии представляют показатели конверсии каждого менеджера, легенда содержит список менеджеров с цветовой дифференциацией.

## Фильтры

**Менеджеры-гр. Фильтры.png** - Панель фильтрации данных

{INTERFACE} Система фильтрации включает выбор года для просмотра данных за конкретный год, выбор месяца для детализации, поиск по офису для фильтрации менеджеров по [офису](#настройки-офисы-и-руководители) и поиск по менеджеру для выбора конкретных сотрудников для анализа.
