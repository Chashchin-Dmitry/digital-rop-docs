# 2. Графики - Оценка

Раздел **Графики - Оценка** предоставляет комплексный анализ качества работы [менеджеров](#аналитика-менеджеры) через визуализацию данных о [звонках](#аналитика-коммуникации) и выполнении [чек-листов](#настройки-скрипты-и-промты). Включает мониторинг загрузки системы и детальную аналитику по этапам скриптов.

## Общий вид

**Оценка.  Общий вид.png** - Основной экран с двумя блоками визуализации

{INTERFACE} Страница включает верхний график "Всего звонков" с динамикой поступления и обработки звонков, нижний блок "Пункты чек-листа" с детализацией выполнения этапов скрипта, а также панель фильтров для настройки отображения данных по периодам и офисам.

## График "Всего звонков"

**Оценка. график "Всего звонков и отработано звонков(системой)".png** - Мониторинг обработки звонков

{TECHNICAL} Линейный график отображает ежедневную статистику по дням месяца (ось X) и количеству звонков (ось Y). Показывает два показателя: "Всего звонков" (поступившие в систему) и "Отработано звонков" (проанализированные системой). Позволяет отслеживать загрузку системы, пики активности и эффективность обработки.

## Анализ по пунктам чек-листа

**Оценка. процентное соотношение выполнение и отклонение по скрипту по этапам по менеджерам.png** - Детализация выполнения этапов

{INTERFACE} Столбчатая диаграмма показывает процентное соотношение выполнения и отклонения по каждому этапу скрипта в разрезе менеджеров. Каждый столбец представляет этап чек-листа, цветовая кодировка отражает уровень соблюдения для быстрого выявления проблемных областей в работе сотрудников.

## Фильтрация данных

**Оценка. Фильтр по офисам.png** - Настройка отображения по подразделениям

{INTERFACE} Фильтр по офисам позволяет анализировать данные конкретных подразделений компании. При выборе офиса все графики автоматически обновляются, показывая статистику только по менеджерам выбранного офиса для более точного анализа эффективности работы.
