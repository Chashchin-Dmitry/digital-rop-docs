# 2. Графики - Статус тестов

Раздел **Графики - Статус тестов** обеспечивает визуальный контроль распределения [тестов](#аналитика-тесты) по статусам и мониторинг процесса обучения [менеджеров](#аналитика-менеджеры). Позволяет руководителям отслеживать эффективность системы тестирования и выявлять узкие места в процессе одобрения.

## Общий вид

**Статус тестов. Общий вид.png** - Основной экран с графиком распределения тестов

{INTERFACE} Раздел содержит круговую диаграмму распределения тестов по статусам, панель фильтров для настройки отображения по временным периодам, офисам и менеджерам, а также детализированную статистику по каждому статусу теста.

## График статусов тестов

**Статус тестов. График кол-ва тестов по статусам .png** - Круговая диаграмма распределения

{TECHNICAL} Круговая диаграмма показывает процентное распределение тестов по трем основным статусам: "Не одобрен" (созданные системой, ожидающие проверки), "Не пройден" (одобренные руководителем, отправленные менеджеру) и "Пройден" (выполненные сотрудниками). Каждый сегмент подписан количеством и процентом от общего числа тестов.

## Фильтрация данных

**Статус тестов. Фильтр по менеджерам и офису.png** - Панель настройки отображения

{INTERFACE} Система фильтрации включает выбор временного периода, фильтрацию по [офисам](#настройки-офисы-и-руководители) компании и конкретным [менеджерам](#аналитика-менеджеры). При изменении фильтров диаграмма автоматически обновляется, показывая актуальную статистику для выбранных параметров.
