# 4. Другое

Раздел **Другое** содержит вспомогательные инструменты и статистику системы: мониторинг обработки файлов и [чат с ИИ](#чат-с-ии) для анализа звонков. Обеспечивает технический контроль работы системы и дополнительные аналитические возможности.

## Статистика файлов

**Статистика файлов. Общий вид.png** - Мониторинг загрузки и обработки аудиофайлов

{TECHNICAL} График "Обработано файлов" отображает ежедневную статистику по дням месяца (ось X) и количеству файлов (ось Y). Показывает два показателя: "Всего файлов" (поступившие через API) и "Файлов обработано" (проанализированные системой). Навигация включает выбор года и месяца для детального анализа и возможность отслеживания производительности системы.

## Сводная таблица по месяцам

**Статистика файлов. Кол-во файлов отработанных системой по месяцам.png** - Сводная статистика за год

{INTERFACE} Табличное представление включает колонки "Месяц" с названием месяца, "Количество файлов" с числом обработанных файлов за месяц и "Итого" с общей суммой за выбранный год. Применяется для анализа сезонности и планирования нагрузки системы.

## Чат с ИИ

**Чат с ИИ. Пример ответа.png** - Интерактивный анализ звонков с искусственным интеллектом

{INTERFACE} Инструмент "Чат с ИИ" позволяет задавать произвольные вопросы о звонках и получать детальные ответы от искусственного интеллекта. Включает поле для ввода вопроса, кнопку "Отправить" для отправки запроса и область отображения ответа ИИ с форматированным текстом.

{TECHNICAL} Система анализирует содержимое звонков на основе заданного вопроса и предоставляет структурированный ответ. Может отвечать на вопросы о соблюдении скриптов, качестве обслуживания, конкретных фразах в разговоре и других аспектах коммуникации. Используется для глубокого анализа отдельных звонков и получения инсайтов по работе менеджеров.
