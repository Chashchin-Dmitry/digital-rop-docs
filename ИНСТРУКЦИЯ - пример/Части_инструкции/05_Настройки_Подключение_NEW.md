# 5. Настройки - Подключение

Раздел для настройки интеграции с корпоративной CRM системой.

## Общий вид

**Подключение. Технические параметры для отправки аудио в систему.png**

{INTERFACE} Страница содержит основные параметры для интеграции с CRM: URL для отправки файлов, API-ключ системы и настройка минимального времени звонка. Доступ к разделу имеют только администраторы системы.

## Основные настройки

### URL для отправки файлов
{INTERFACE} Отображается готовый URL для настройки в CRM. Этот адрес используется для отправки аудиофайлов из корпоративной системы в "Цифровой РОП". URL содержит базовый адрес и указание на необходимость API-ключа.

### API-ключ системы  
{TECHNICAL} Уникальный ключ доступа для безопасной интеграции между системами. Ключ используется для аутентификации всех запросов из CRM. Ключ можно скопировать из интерфейса для настройки в корпоративной системе. Подробное техническое описание API доступно в разделе [API документация](#api-документация).

### Минимальное время звонка
{INTERFACE} Поле для установки минимальной длительности звонка в секундах. Звонки короче указанного времени не будут обрабатываться системой. По умолчанию значение установлено в 0 (обрабатываются все звонки).

## Принцип работы интеграции

{TECHNICAL} CRM отправляет ссылки на аудиофайлы вместе с метаданными звонка. Система "Цифровой РОП" скачивает файлы по переданным ссылкам и автоматически ставит их в очередь на обработку. После анализа результаты отправляются обратно в CRM через настроенные webhook.

## Параметры для CRM

### Обязательные поля
{INTERFACE} При настройке интеграции в CRM необходимо передавать следующие данные:
- **call_id**: Уникальный идентификатор звонка
- **audio_url**: Прямая ссылка на аудиофайл для скачивания  
- **manager_id**: Идентификатор менеджера в CRM
- **deal_id**: Идентификатор сделки

### Необязательные поля
{TECHNICAL} Дополнительные параметры для расширенной аналитики:
- **deal_stage_id**: Идентификатор этапа сделки
- **stage_id**: Идентификатор воронки продаж
- **duration**: Длительность звонка в секундах

## Безопасность и администрирование

{INTERFACE} Раздел доступен только администраторам системы. Менеджеры и руководители офисов не видят технические настройки подключения. API-ключ генерируется один раз и не может быть изменен через интерфейс.

### Важные примечания
- Сохраните API-ключ в надежном месте
- URL нельзя изменить после настройки интеграции  
- Готовые параметры предоставляются для удобства настройки CRM

## Результаты обработки

{TECHNICAL} После анализа звонка система отправляет результаты обратно в CRM через два типа уведомлений:
1. **Результаты чек-листа** - оценка соблюдения корпоративного скрипта
2. **Результаты аналитики** - данные пользовательских таблиц

Время отправки результатов составляет 5-15 минут после загрузки аудиофайла.
