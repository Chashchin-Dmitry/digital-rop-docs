## Поиск пользователей

**Пользователи. Работа поиска пользователей.png** - Поиск по ФИО и email

{INTERFACE} Введите текст в поле поиска для фильтрации списка пользователей. Поиск работает по всем полям: имя, фамилия, отчество, email. Результаты отображаются мгновенно.

## Редактирование пользователя

**Пользователи. Редактирование пользвоателя на кнопку изменить.png** - Форма изменения данных пользователя

{INTERFACE} Нажмите "Изменить" рядом с пользователем. Откроется форма с полями: ФИО, email, роль в системе (выпадающий список), пароль (опционально), статус ВКЛ/ВЫКЛ. Нажмите "Сохранить" для применения изменений.

## Выбор ролей в системе

**Пользователи. Выбор прав.png** - Назначение роли пользователю

{TECHNICAL} Доступные роли: "Менеджер" (только свои звонки), "Руководитель офиса" (данные офиса + одобрение тестов), "Администратор" (полный доступ к системе). Каждому пользователю назначается одна роль, которая определяет уровень доступа.

## Интеграция с LDAP

{TECHNICAL} Система использует LDAP для аутентификации пользователей через учетные данные Setl Group. Пользователи входят с корпоративным логином и паролем без дополнительной регистрации. Активные пользователи имеют email и статус "ВКЛ".
