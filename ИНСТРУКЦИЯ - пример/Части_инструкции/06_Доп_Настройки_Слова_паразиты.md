# 6. Дополнительные настройки - Слова-паразиты

Раздел для управления словарем нежелательных слов и выражений, которые система отслеживает и выделяет в транскриптах звонков.

## Общий вид

**Слова-паразиты. Общий вид.png** - Управление словарем нежелательных слов

{INTERFACE} Страница содержит поле ввода для добавления новых слов, кнопку "Добавить слово", большое текстовое поле "Набор слов паразитов" со списком отслеживаемых слов и кнопку "Обновить словарь" для сохранения изменений.

## Принцип работы

{TECHNICAL} Система автоматически сканирует транскрипты всех звонков и выделяет красным цветом слова из настроенного словаря. Выделение происходит только в речи менеджера, а не клиента. Словарь используется для контроля качества речи и профессионализма общения.

## Добавление слов

{INTERFACE} Введите слово в поле ввода и нажмите "Добавить слово" для включения в словарь. Можно также редактировать весь список в текстовом поле "Набор слов паразитов" и нажать "Обновить словарь" для сохранения изменений.

## Примеры слов-паразитов

Типичные категории для добавления в словарь:
- **Паразиты речи**: "э-э-э", "а-а-а", "как бы", "в общем", "типа"
- **Слова неуверенности**: "может быть", "наверное", "вроде как"
- **Нецензурная лексика** и непрофессиональные выражения
- **Повторы**: "да-да-да", "хорошо-хорошо"
