# Часто задаваемые вопросы (FAQ)

## Вопросы по входу и доступу

### Не могу войти в систему

**Q: Не получается войти в систему, что делать?**

A: Проверьте следующее:
1. **Правильность URL** - используйте **http://10.28.32.81/**
2. **Логин и пароль** - это те же данные, что вы используете для CRM Setl Group
3. **LDAP-соединение** - система использует корпоративную аутентификацию
4. **Права доступа** - убедитесь, что вам предоставлен доступ к системе

**Если проблема не решается** - обратитесь к администратору системы.

### Забыл пароль

**Q: Забыл пароль от системы**

A: Система использует **LDAP-интеграцию** с корпоративной CRM. Для восстановления пароля:
1. Обратитесь к IT-службе Setl Group
2. Восстановите пароль от CRM
3. Используйте восстановленные данные для входа в "Цифровой РОП"

### Ограниченный доступ

**Q: Не вижу данные других менеджеров/офисов**

A: Это нормальное поведение системы:
- **Менеджеры** видят только свои данные
- **Руководители** видят данные только своего офиса  
- **Администраторы** имеют полный доступ

Если нужны дополнительные права - обратитесь к администратору.

## Вопросы по оценкам и анализу

### Непонятная оценка ИИ

**Q: Не согласен с оценкой ИИ, кажется неправильной**

A: Для разбора ситуации:
1. **Изучите комментарии** - ИИ объясняет каждую оценку
2. **Прослушайте звонок** - сравните с транскрипцией
3. **Проверьте критерии** - ознакомьтесь с чек-листом
4. **Используйте кнопку "Комментарий к оценке"** - оставьте обратную связь
5. **Обратитесь к руководителю** - для совместного разбора

### Странная транскрипция

**Q: В транскрипции есть ошибки или непонятные слова**

A: Качество транскрипции зависит от:
- **Качества связи** во время звонка
- **Четкости речи** участников
- **Фонового шума** 

**Система автоматически**:
- Делает повторную обработку при подозрении на ошибки
- Проверяет транскрипцию на наличие нецензурной лексики
- Использует наиболее точный вариант

### Нет оценки по звонку

**Q: Звонок есть, но оценки нет**

A: Возможные причины:
1. **Звонок еще обрабатывается** - процесс занимает 5-15 минут
2. **Ошибка транскрибации** - плохое качество аудио
3. **Технические проблемы** - обратитесь к администратору

Обычно обработка завершается в течение 15 минут после поступления звонка.

## Вопросы по тестам

### Неправильный тест

**Q: Тест кажется неправильным или некорректным**

A: Тесты создаются автоматически, но проходят проверку:
1. **Изучите связанный звонок** - тест основан на реальных ошибках
2. **Обратитесь к руководителю** - он может отредактировать тест
3. **Пройдите тест** - изучите объяснения правильных ответов

Руководители могут корректировать тесты перед утверждением.

### Нет новых тестов

**Q: Давно не получаю новые тесты**

A: Тесты создаются только при наличии ошибок:
- **Хорошая работа** - если нет этапов с -1 баллом, тесты не генерируются
- **Ожидание утверждения** - руководитель еще не утвердил созданные тесты
- **Проверьте фильтры** - убедитесь, что не установлены ограничивающие фильтры

### Не могу пройти тест

**Q: Не открывается тест для прохождения**

A: Проверьте:
1. **Статус теста** - должен быть "Утвержден"
2. **Права доступа** - тест должен быть назначен именно вам
3. **Техническое состояние** - попробуйте обновить страницу

## Технические проблемы

### Система медленно работает

**Q: Система тормозит, долго загружается**

A: Возможные решения:
1. **Обновите страницу** - используйте F5 или Ctrl+R
2. **Очистите кэш браузера**
3. **Проверьте интернет-соединение**
4. **Используйте рекомендуемые браузеры** - Chrome, Firefox
5. **Закройте лишние вкладки** - освободите память

### Не отображаются изображения

**Q: Не видно скриншотов или изображений**

A: Попробуйте:
1. **Обновить страницу**
2. **Проверить блокировщики рекламы** - могут блокировать изображения
3. **Использовать другой браузер**
4. **Очистить кэш и cookies**

### Ошибка при экспорте

**Q: Не получается скачать отчет в Excel**

A: Решения:
1. **Уменьшите период** - выберите меньший диапазон дат
2. **Проверьте права доступа** - функция может быть ограничена по роли
3. **Попробуйте позже** - сервер может быть перегружен
4. **Обратитесь к администратору** - для решения технических проблем

## Вопросы по функциям

### Не работает фильтр

**Q: Фильтры не применяются или работают неправильно**

A: Проверьте:
1. **Корректность выбора** - убедитесь, что выбраны правильные параметры
2. **Наличие данных** - возможно, нет данных по выбранным критериям
3. **Права доступа** - некоторые фильтры доступны только определенным ролям
4. **Обновление страницы** - иногда помогает перезагрузка

### Не находит нужные данные

**Q: Не могу найти конкретный звонок или данные**

A: Система пока не имеет глобального поиска:
1. **Используйте фильтры** - по дате, менеджеру, офису
2. **Проверьте период** - возможно, данные за другую дату
3. **Уточните права доступа** - можете видеть не все данные
4. **Обратитесь к руководителю** - за помощью в поиске

## Вопросы администраторов

**Q: Звонки не поступают из CRM**

A: Проверьте:
1. **API ключи** - корректность настройки в разделе "Подключение"
2. **URL эндпоинтов** - правильность адресов для отправки
3. **Формат данных** - соответствие требуемым полям
4. **Логи системы** - для выявления ошибок интеграции
