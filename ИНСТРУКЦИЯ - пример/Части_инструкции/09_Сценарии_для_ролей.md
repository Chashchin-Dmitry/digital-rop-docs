## Решение типичных проблем

### Проблемы менеджера

**"Не понимаю оценку ИИ"**:
1. Внимательно прочитать комментарии к оценке
2. Прослушать оригинальный звонок
3. Сравнить с критериями в чек-листе
4. Обратиться к руководителю за разъяснениями

**"Тест кажется неправильным"**:
1. Изучить связанный звонок
2. Понять контекст создания теста  
3. Обратиться к руководителю для корректировки
4. Пройти тест и изучить объяснения

### Проблемы руководителя

**"Менеджер не согласен с оценкой"**:
1. Совместный анализ спорного звонка
2. Разбор критериев оценки
3. При необходимости - корректировка чек-листа
4. Добавление комментария к оценке

**"Низкое качество автоматических тестов"**:
1. Анализ промтов для генерации тестов
2. Обращение к администратору для корректировки
3. Более тщательное редактирование перед утверждением

### Проблемы администратора

**"Система оценивает неточно"**:
1. Анализ примеров неточных оценок
2. Корректировка промтов для оценки
3. Тестирование на контрольной выборке
4. Постепенное внедрение улучшений

**"Пользователи жалуются на интерфейс"**:
1. Сбор конкретной обратной связи
2. Анализ наиболее частых проблем
3. Обучение пользователей возможностям системы
4. Передача предложений команде разработки

## Взаимодействие между ролями

### Эскалация проблем

**Менеджер → Руководитель**:
- Спорные оценки звонков
- Неясные тесты
- Технические проблемы

**Руководитель → Администратор**:
- Проблемы с чек-листами
- Неточность промтов
- Запросы на новую аналитику

**Администратор → Поддержка**:
- Технические неисправности
- Запросы на доработку функций
- Проблемы интеграции

### Обучение и развитие

**Обмен опытом**:
- Руководители делятся успешными практиками
- Администраторы обучают работе с новыми функциями
- Менеджеры дают обратную связь по качеству тестов

**Непрерывное улучшение**:
- Регулярный анализ эффективности системы
- Корректировка процессов на основе опыта
- Внедрение лучших практик во всех офисах

**Важно**: Успешное использование системы зависит от четкого понимания каждой ролью своих задач и активного взаимодействия между всеми участниками процесса.
