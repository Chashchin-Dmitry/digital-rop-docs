# Словарь терминов

## Основные термины системы

### Цифровой РОП
**Определение**: Система автоматической оценки телефонных звонков с использованием искусственного интеллекта, развернутая для Setl Group.

**Применение**: Основное название системы, используемое во всей документации и интерфейсе.

### Промт (Prompt)
**Определение**: Запрос или инструкция для искусственного интеллекта, объясняющая, как нужно анализировать звонки и какие результаты выдавать.

**Типы промтов**:
- Промты для оценки этапов скрипта
- Промты для генерации тестов
- Промты для кастомной аналитики таблиц
- Промты для определения конверсий

### Чек-лист (Скрипт)
**Определение**: Набор этапов и критериев, по которым искусственный интеллект оценивает качество выполнения телефонного звонка менеджером.

**Структура**: Включает название этапа, критерии выполнения, условия игнорирования и уровень строгости оценки.

### Транскрипция
**Определение**: Текстовое представление аудиозаписи звонка, созданное автоматически системой распознавания речи.

**Особенности**: Содержит временные метки, разделение по спикерам и проходит двойную проверку на качество.

### Диаризация
**Определение**: Процесс автоматического разделения аудиозаписи на сегменты по говорящим (Спикер 0, Спикер 1).

**Цель**: Определить, кто именно говорит в каждый момент времени для корректного анализа.

## Технические термины

### LDAP
**Определение**: Lightweight Directory Access Protocol - протокол для доступа к службе каталогов, используемый для интеграции с корпоративными учетными записями.

**В системе**: Обеспечивает единый вход для пользователей Setl Group без дополнительной регистрации.

### API (Application Programming Interface)
**Определение**: Интерфейс программирования приложений - набор методов и протоколов для взаимодействия между системами.

**В системе**: Используется для получения звонков из CRM и отправки результатов обратно.

### Webhook
**Определение**: HTTP-обратный вызов, метод автоматической отправки данных из одной системы в другую при наступлении определенных событий.

**В системе**: Автоматическая отправка результатов анализа и оценок в CRM Setl Group.

### ИИ (Искусственный интеллект)
**Определение**: Технология машинного обучения, используемая для анализа текста, понимания контекста и принятия решений на основе заданных критериев.

**Функции в системе**: Оценка звонков, генерация тестов, извлечение аналитики, анализ конверсий.

## Бизнес-термины

### Конверсия
**Определение**: Достижение заранее определенной цели в результате телефонного звонка (запись на встречу, получение контактов, заключение сделки).

**Настройка**: Определяется через специальные промты в разделе настроек.

### Воронка продаж
**Определение**: Последовательность этапов, через которые проходит потенциальный клиент от первого контакта до совершения покупки.

**В системе**: Чек-листы могут быть настроены под разные стадии воронки.

### CRM (Customer Relationship Management)
**Определение**: Система управления взаимоотношениями с клиентами, используемая в Setl Group.

**Интеграция**: Источник звонков и получатель результатов анализа.

### KPI (Key Performance Indicators)
**Определение**: Ключевые показатели эффективности, используемые для оценки работы менеджеров.

**В системе**: Рейтинги, количество баллов, процент прохождения тестов, показатели конверсии.

## Термины интерфейса

### Спикер 0 / Спикер 1
**Определение**: Условное обозначение участников разговора в транскрипции.

**Обычно**: Спикер 0 - менеджер, Спикер 1 - клиент (может варьироваться в зависимости от звонка).

### Баллы (+1/-1)
**Определение**: Система оценки выполнения этапов скрипта.
- **+1 балл** - этап выполнен согласно критериям
- **-1 балл** - этап не выполнен или выполнен неправильно

### Процентная оценка
**Определение**: Точное измерение качества выполнения этапа от 0% до 100%, учитывающее уровень строгости настроек.

### Пагинация
**Определение**: Разделение больших объемов данных на страницы для удобного просмотра.

**В системе**: Используется в списках звонков, тестов и других данных.

## Роли и права

### Супер-администратор
**Определение**: Роль с максимальными правами доступа ко всем функциям системы.

**Возможности**: Полный доступ к данным, настройкам, управлению пользователями.

### Администратор
**Определение**: Роль с расширенными правами для настройки системы и управления процессами.

**Отличие от супер-админа**: Может иметь ограничения в доступе к некоторым данным или функциям.

### Руководитель офиса
**Определение**: Роль для управления командой конкретного офиса или подразделения.

**Ограничения**: Доступ только к данным своего офиса, управление тестами для подчиненных.

### Менеджер
**Определение**: Базовая роль для сотрудников, ведущих телефонные продажи.

**Доступ**: Только к собственным данным, звонкам и тестам.

## Сокращения и аббревиатуры

### URL
**Расшифровка**: Uniform Resource Locator
**Определение**: Веб-адрес системы или ресурса в интернете.

### HTTP
**Расшифровка**: HyperText Transfer Protocol  
**Определение**: Протокол передачи данных в интернете.

### JSON
**Расшифровка**: JavaScript Object Notation
**Определение**: Формат обмена данными между системами.

### IP
**Расшифровка**: Internet Protocol
**Определение**: Протокол интернета, в контексте системы - IP-адрес сервера (10.28.32.81).

## Специфические термины Setl Group

### Термины CRM Setl Group
**Определение**: Специфические поля, процессы и обозначения, используемые в корпоративной CRM системе Setl Group.

**Примеры**: Этапы воронки продаж, коды сделок, статусы клиентов, внутренние классификаторы.

### Организационная структура
**Определение**: Иерархия подразделений, офисов и ролей в компании Setl Group.

**Компоненты**: Названия офисов, структура подчинения, бизнес-процессы, зоны ответственности.

### Продуктовые термины
**Определение**: Специализированная терминология, связанная с продуктами и услугами Setl Group.

**Область применения**: Названия продуктов, этапы продаж, отраслевые стандарты, клиентские сегменты.

## Слова-паразиты

**Определение**: Ненужные слова и звуки в речи (например, "эм", "ну", "типа"), которые система может выделять и анализировать.

**Настройка**: Доступна в разделе "Доп. настройки → Слова-паразиты" для администраторов.

**Функция**: Автоматическое выделение в транскрипции для анализа качества речи менеджера.

## Дополнительные термины

### Тайм-код (временная метка)
**Определение**: Указание точного времени в аудиозаписи, когда была произнесена определенная фраза.

**Формат**: Обычно в формате ЧЧ:ММ:СС (часы:минуты:секунды) от начала звонка.

### Эскалация
**Определение**: Процесс передачи проблемы или вопроса на более высокий уровень управления.

**В системе**: Обращение от менеджера к руководителю, от руководителя к администратору.

### Фильтрация
**Определение**: Функция отбора данных по определенным критериям (дате, менеджеру, офису, статусу).

**Цель**: Сужение объема отображаемой информации для более удобного анализа.

**Примечание**: Данный словарь может быть дополнен и уточнен по мере получения дополнительной информации от команды Setl Group и в процессе использования системы.
