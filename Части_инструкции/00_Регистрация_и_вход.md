# Регистрация и вход в систему

**URL платформы:** https://rop.bvmax.ru/login

Регистрация в системе "Цифровой РОП" осуществляется через Telegram-бота. Это обеспечивает безопасность и удобство дальнейшей работы — через того же бота менеджеры получают тесты и отчёты.

## Шаг 1: Откройте страницу регистрации

**Регистрация. Страница входа.png**

Перейдите по адресу https://rop.bvmax.ru/login. На экране вы увидите форму входа и кнопку регистрации.

{INTERFACE} Нажмите кнопку "Регистрация" для начала процесса создания аккаунта. На следующем экране будет подробная инструкция по подключению Telegram-бота.

## Шаг 2: Создайте группу в Telegram

**Регистрация. Экран регистрации.png**

Система объясняет процесс регистрации: необходимо создать группу в Telegram, добавить в неё бота и назначить его администратором.

**Регистрация. Создать группу с ботом.png**

{INTERFACE} Создайте новую группу в Telegram. Название может быть любым — например, "Цифровой РОП - [Название компании]". Добавьте в группу бота **@BVM_Assistant_Bot**.

## Шаг 3: Назначьте бота администратором

Для корректной работы бот должен быть администратором группы.

**Регистрация. Назначить бота админом шаг 1.png**

{INTERFACE} Откройте настройки группы, нажав на её название в верхней части экрана.

**Регистрация. Назначить бота админом шаг 2.png**

{INTERFACE} Перейдите в раздел "Администраторы".

**Регистрация. Назначить бота админом шаг 3а.png**

**Регистрация. Назначить бота админом шаг 3б.png**

{INTERFACE} Нажмите "Добавить администратора" и выберите бота @BVM_Assistant_Bot из списка участников группы.

**Регистрация. Назначить бота админом шаг 4.png**

{INTERFACE} Сохраните настройки администратора. Права можно оставить по умолчанию.

## Шаг 4: Получите код активации

**Регистрация. Команда register.png**

{INTERFACE} В группе с ботом напишите команду `/register`. Бот ответит сообщением с уникальным кодом активации.

**Регистрация. Получить код.png**

{TECHNICAL} Код активации — это уникальный идентификатор, привязывающий вашу группу к аккаунту в системе. Код действителен ограниченное время.

## Шаг 5: Активируйте группу на сайте

**Регистрация. Вставить код.png**

{INTERFACE} Скопируйте полученный код и вставьте его в поле "Код активации" на странице регистрации. Нажмите кнопку "Проверить код активации".

**Регистрация. Успешная активация.png**

При успешной активации появится зелёная плашка с подтверждением и названием вашей группы.

## Шаг 6: Заполните данные аккаунта

**Регистрация. Ввести данные.png**

{INTERFACE} Заполните форму регистрации:
- **Имя** — ваше имя
- **Фамилия** — ваша фамилия
- **Email** — адрес электронной почты (будет использоваться как логин)
- **Пароль** — придумайте надёжный пароль

## Шаг 7: Выберите интеграцию

**Регистрация. Выбор интеграции.png**

{INTERFACE} Выберите источник данных — CRM-систему, облачное хранилище или телефонию:

**CRM-системы:**
- **Битрикс24** — интеграция с облачным и коробочным Битрикс24
- **AmoCRM** — интеграция с AmoCRM
- **MacroCRM** — интеграция с MacroCRM

**Облачные хранилища:**
- **Яндекс Диск** — загрузка записей звонков из Яндекс Диска

**Телефонии:**
- **Новофон** — интеграция с телефонией Новофон
- **МТС Автосекретарь** — интеграция с МТС Автосекретарь

{TECHNICAL} После выбора интеграции система предоставит инструкции по подключению. Для каждой интеграции процесс настройки отличается — следуйте инструкциям на экране. Если вашей системы нет в списке — обратитесь в поддержку, возможно добавление новых интеграций.

## Вход в систему

После завершения регистрации вы можете входить в систему по адресу https://rop.bvmax.ru/login, используя email и пароль, указанные при регистрации.

## Важные замечания

- Группа в Telegram используется для всей компании — в неё нужно добавить всех сотрудников, которые будут работать с системой
- Менеджеры должны **начать диалог с ботом** (нажать /start в личных сообщениях), чтобы получать тесты
- Один аккаунт = одна компания. Для разных компаний нужны разные аккаунты
